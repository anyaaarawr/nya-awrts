WELCOME TO NYAAWRTS

cat-inspired website

Nya Nya Art is a cat-themed website focused on creativity and simple web design. It was created as a personal project to learn how websites are built using HTML, CSS, and JavaScript. The design is inspired by cats, especially their cute and playful “nya” style. The website is not just about cats, but also about the process of building it and learning how to code.
